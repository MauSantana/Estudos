Colaboração e União na vizinhança para um bem maior

O “Vai ali pra mim” é uma plataforma onde pessoas que precisam de ajuda para comprar algo, encontram pessoas dispostas a ajudar. Nosso intuito é promover uma maior integração entre moradores proximos um ao outro onde pessoas com uma maior disponibilidade e/ou possibilidade possam atráves de um gesto de solidariedade, realizar tarefas para pessoas que não podem fazê-las no momento.

Como Funciona?

Seja Entregador ou Consumidor

O entregador, pessoa que realizará a boa ação, fará um anuncio assim que for realizar alguma tarefa e se colocará a disposição para fazer o mesmo pelo seus vizinhos que solicitarem ajuda. Tudo isso através de uma plataforma simples e acessível.

Alem de uma boa ação, receba remuneração

Além de poder contribuir com o bem-estar de uma pessoa que necessita de ajuda, o entregador poderá solicitar um pequeno taxa de entrega para cada tarefa que realizar.
